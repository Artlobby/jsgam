A Point & Click Adventure Game Engine

Introduction

JavaScript Graphic Adventure Maker (JSGAM) is a set of tools to make adventure games inspired by the 90s point and click games running directly in the web browser.
The project started to make personal games but I decided to share it so anyone with the same interest for classic adventure games can make them quickly without spend time in the engine.
To make games with JSGAM you don't need any knowledges about programming nor design, but It'll be easier if you know at least the basics of GIMP, Blender, HTML, CSS and JavaScript.

Setting up the tools

JSGAM-Tools GIMP plug-in

Download

Copy the jsgam-tools.py should the GIMP plug-ins folder which is located in:

  • Linux: ~/.config/GIMP/2.10/plug-ins
  • Windows: C:\Users\yourname\AppData\Roaming\GIMP\2.10\plug-ins
You should find it under File>Export to JSGAM... after restarting GIMP.

Usage

  • The bottom layer isn't exported because the script takes it as the background.
  • The bottom path is exported as WalkArea and the other paths are considered Obstacles.
  • Add 'obj' to the name path to create an invisible object instead of an obstacle.

Coatools

Blender Coa-Tool

GIMP Exporter:
The coatools_exporter.py should be copied to your GIMP plug-ins folder which is located in:

  • Linux: ~/.config/GIMP/2.10/plug-ins
  • Windows: C:\Users\yourname\AppData\Roaming\GIMP\2.10\plug-ins

Blender plug-in:
Zip the coa_folder. Go to File -> User Preferences -> Add-ons and click the Install from file... button. This will install and enable the Addon for Blender. Don’t forget to save the user preferences, otherwise the addon will not be activated after restart.
